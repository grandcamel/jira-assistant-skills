# Browse JIRA Skills

Display a categorized list of all available JIRA automation skills.

## Instructions

Read and present the skill catalog from the marketplace skill. Group skills by category and show:
- Skill name
- Brief description
- Example use case

## Output Format

Present as a formatted table or list showing:

### Core Operations
- **jira-issue** - Create, read, update, delete issues
- **jira-lifecycle** - Workflow transitions and status changes
- **jira-search** - JQL queries and saved filters

### Collaboration
- **jira-collaborate** - Comments, attachments, watchers
- **jira-relationships** - Issue linking and dependencies

### Agile & Planning
- **jira-agile** - Sprints, epics, backlog management
- **jira-time** - Time tracking and estimates

### Bulk & Automation
- **jira-bulk** - Mass operations with dry-run support
- **jira-dev** - Git branch names, commit parsing

### Administration
- **jira-fields** - Custom field discovery
- **jira-ops** - Cache and performance utilities
- **jira-admin** - Project administration

### Service Management
- **jira-jsm** - Service desk, SLAs, queues

### Meta
- **jira-assistant** - Routes to the right skill automatically

For detailed info on any skill, use: `/skill-info <skill-name>`
